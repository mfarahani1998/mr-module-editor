# Roadmap

This roadmap keeps the project incremental and demoable.

## Milestone A — Baseline Handoff

Phase 1: Demo Freeze, Baseline Docs, and Handoff Readiness.

Goal: the current demo can be run and understood by someone else.

## Milestone B — Extension Seam

Phase 2: Step Catalog v0.2 and Modular Validation.

Goal: adding steps no longer requires core/editor edits in many places.

## Milestone C — Generic MR Learning Toolkit

Phase 3 + Phase 4.

Goal: a small non-robotics module can be authored and previewed without JSON editing.

## Milestone D — MR Layout Maturity

Phase 5.

Goal: head/world/object layout is robust enough for repeated demos.

## Milestone E — First Domain Expansion

Phase 6.

Goal: a non-robotics domain pack proves platform extensibility.

## Milestone F — Portable Modules

Phase 7.

Goal: modules can be exported/imported/packaged reliably.

## Milestone G — Longer Modules

Phase 8.

Goal: sections/storyboard and flow validation support larger self-learning modules.

## Milestone H — Platform Preparation

Phase 9.

Goal: schema migration, compatibility validation, runtime services, and target profiles prepare the project for larger platform decisions.

## Things not to prioritize yet

- Full standalone authoring app.
- Full graph editor.
- Full visual timeline.
- Persistent spatial anchors.
- Collaboration/cloud backend.
- Deep RoboticsLite expansion.
